# Dewbwah Marketing

Marketing for contractors that don't cut corners. We build systems that generate predictable local jobs without chasing the next one.

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Package Manager:** pnpm

## Available Scripts

- `pnpm dev` - Start development server
- `pnpm build` - Build for production
- `pnpm start` - Start production server
- `pnpm lint` - Run ESLint

## Pages

- **Home** — Hero video, services overview, problem/solution section, CTA form
- **About** — Team bios, values, company timeline, FAQ
- **Services** — Main overview + sub-pages (Paid Ads, SEO, Web Design, Reputation)
- **Work** — Case studies with metrics and testimonials
- **Blog** — Coming soon placeholder with preview cards
- **Contact** — Contact form, info, embedded map, FAQ
